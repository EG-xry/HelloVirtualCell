# 虚拟细胞可视化工具 · 使用指南

## 快速开始

直接用浏览器打开 `artifacts/cell_visualization.html`，无需安装任何依赖，离线可用。

首次打开后页面自动进入**对照组（Control）**状态并开始动画。

## 细胞结构说明

主视图绘制了一个哺乳动物细胞的剖面，各结构均有动态指示：

| 结构 | 视觉特征 | 动态行为 |
|------|---------|---------|
| **细胞膜** | 椭圆形边框 + 短线标注 | 膜受体根据扰动条件激活（绿色闪烁） |
| **细胞核** | 中心双层椭圆，内有染色质 | 核孔随基因表达活性开闭；核仁可见 |
| **线粒体** | 4个豆形结构，内有内嵴 | 活性高时橙色增亮，ATP指示点闪烁 |
| **内质网** | 核周放射状曲线 | ER应激时出现红色粒子涌现 |
| **高尔基体** | 右上角叠层结构 | 活性高时可见囊泡出芽 |
| **核糖体** | 胞质内散布小点 | 翻译活跃时亮度上升 |

**粒子流动**含义：
- 从细胞核向外辐射 → 基因表达产物（mRNA / 蛋白质）
- 线粒体内循环粒子 → 氧化磷酸化 / ATP合成
- 红色粒子 → ER应激信号
- 从细胞膜向内的绿色粒子 → 受体介导的胞外信号转导

## 切换扰动条件

底部五个按钮对应五种实验条件，点击后所有面板同步过渡（约1.2秒动画）：

| 按钮 | 对应扰动 | 主要变化 |
|------|---------|---------|
| **对照** | 正常细胞 | 所有通路处于基础活性，粒子流均匀 |
| **基因A敲除** | Ldha / Pkm 敲除 | 糖酵解基因（前4个）骤降；TCA基因代偿性上调；线粒体变暗 |
| **基因B敲除** | Cs / Aco2 敲除 | 糖酵解基因上调；TCA基因（第9–12个）骤降；ER出现红色应激粒子 |
| **药物X** | 受体激动剂 | 全基因组普遍上调；膜受体高频激活；绿色膜信号粒子增多 |
| **细胞因子** | IL-6 / TNFα | 所有基因和通量达到最高；细胞核粒子密度最大；线粒体全亮 |

切换时右侧面板的**基因表达条形图**颜色编码：
- **扰动色条**（青/橙/粉/紫/绿）= 相对对照组上调
- **红色条** = 相对对照组下调
- **灰色条** = 与对照组相近

## 视觉风格切换

右上角三个风格按钮，切换不同配色方案，不影响数据：

| 风格 | 适用场景 |
|------|---------|
| **精密仪器** | 默认，青蓝色系，暗室演示或截图 |

| **赛博荧光** | 霓虹绿/品红，适合海报或会议展板 |

## 读取代谢通量图（右下角）

12条柱形对应玩具FBA模型中的12个反应，从左到右依次为：

`HK → PFK → PK → LDH → CS → IDH → SDH → MDH → CI → CII → CIII → ATP`

- **前4条**（HK–LDH）为糖酵解
- **中4条**（CS–MDH）为TCA循环
- **后4条**（CI–ATP）为氧化磷酸化 / 电子传递链

柱高代表预测通量，红色（< 0.3）表示该反应受到抑制。

## 与 Pipeline 的对应关系

可视化数据来自 `openvcell/` 各阶段的仿真输出：

| 可视化元素 | 对应 Pipeline 阶段 |
|-----------|-----------------|
| 细胞核粒子密度 | Stage 2 基因表达嵌入（`stage2_foundation.py`） |
| 基因表达条形图 | Stage 3 扰动预测结果（`stage3_perturb.py`） |
| 代谢通量柱图 | Stage 4 FBA输出（`stage4_mechanism.py`） |
| 线粒体活性 | Stage 4 酶丰度 → 通量上界映射 |

> 当前可视化使用内置仿真参数；如需加载真实 pipeline 输出，可将 `PERTURB_DATA` 中的 `genes` 和 `fluxes` 数组替换为 `artifacts/metrics.json` 中对应数值。

## 常见问题

**动画卡顿** — 浏览器建议使用 Chrome / Edge；Firefox 下 Canvas shadowBlur 性能较低，可接受。

**标签显示不全** — 窗口过小时部分 SVG 标签会超出视口，建议全屏（F11）或最大化窗口使用。

**扰动数据与论文不符** — 当前数据为合成仿真值，仅用于演示通路逻辑；替换为真实实验数据需修改 `PD` 对象中对应条目。
